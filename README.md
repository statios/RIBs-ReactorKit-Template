# RIBs-ReactorKit-Template

# Install
To install the RIBs-ReactorKit Xcode templates, run:

> make install

To uninstall the RIBs-ReactorKit Xcode templates, run:

> make uninstall

# Requirements
- ReactorKit
- RIBs
